| 10<sup>-5</sup> lx | 가장 밝은 별(시리우스)의 빛 |
| 10<sup>-4</sup> lx | 하늘을 덮은 완전한 별빛 |
| 0.002 lx           | 대기광이 있는 달 없는 맑은 밤 하늘 |
| 0.01 lx            | 초승달 |
| 0.27 lx            | 맑은 밤의 보름달 |
| 1 lx               | 열대 위도를 덮은 보름달 |
| 3.4 lx             | 맑은 하늘 아래의 어두운 황혼 |
| 50 lx              | 거실 |
| 80 lx              | 복도/화장실 |
| 100 lx             | 매우 어두운 낮 |
| 320 lx             | 권장 오피스 조명 |
| 400 lx             | 맑은 날의 해돋이 또는 해넘이 |
| 1000 lx            | 인공 조명, 일반적인 TV 스튜디오 조명 |
| 10,000–25,000 lx   | 낮 |
| 32,000–130,000 lx  | 직사광선 |
